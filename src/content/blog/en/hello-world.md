## Hello

This is the first blog post. From here I'll share updates on projects, thoughts on photography, behind-the-scenes moments, and anything else worth telling.

## How it works

Posts are written in Markdown and saved in the `src/content/blog/en/` folder. Each file is an article: the file name becomes the URL slug.

To add a new post, create a `.md` file with the frontmatter:

```md
---
title: "Post title"
date: "2026-01-01"
excerpt: "Brief description for the card and search engines."
tags: ["tag1", "tag2"]
coverImage: "https://res.cloudinary.com/..."  # optional
draft: false
---

Post text here.
```

Set `draft: true` to save a draft without publishing it in production.
